# Sales Analysis Project

This is a simple Python project I made to analyze sales data by country.

The script reads a sales dataset (randomly found online), groups the data by country, and calculates the average total revenue for each country.
After that, it finds the three countries with the lowest average revenue, and creates separate CSV files for those countries,
so it is easier to look at their sales data individually and sort or explore them later.

In my case, the lowest three were Kuwait, Kyrgyzstan, and New Zealand.
