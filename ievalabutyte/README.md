# Export-as-a-Service Market Players Analysis

## Project Overview
A data analysis project researching and comparing companies
in the Export-as-a-Service (EaaS) market.

## Objectives
1. Collect data on EaaS market players
2. Perform exploratory data analysis (EDA)
3. Build visualizations to compare players
4. Apply clustering to segment companies
5. Automate data collection using Python

## Tools
- Python 3.x
- pandas
- matplotlib / seaborn
- scikit-learn
- Jupyter Notebooks
- Git
